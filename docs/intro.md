Quantago is a cloud-native backtesting platform built on Cloudflare Workers, Pages, and durable workflows.

Use these guides to get productive quickly:

- Start with [Quick Start](./quickstart) for local setup and first deploy steps.
- Read [Architecture](./architecture) for the system layout, data flow, and infrastructure model.
- Use [Deployment](./deployment) and [Deployment Checklist](./deployment-checklist) when shipping to production.
- Reference [Strategy Protocol](./strategy-protocol) when building native, remote, or future WASM strategies.
- Review the standards section for backend, component, and Zustand conventions used across the repo.

Platform domains:

- `quantago.co` for the landing site
- `docs.quantago.co` for documentation
- `app.quantago.co` for the trading app
- `admin.quantago.co` for the admin surface
- `api.quantago.co` for the backend API
